Abstract

This project implements a Deposit Management System in C, using binary file handling to manage customer deposit accounts.
The system allows users to manage both Fixed Deposits (FD) and Recurring Deposits (RD) by providing options to add, view, search, update, and delete deposit records.
All account information is stored in deposits.dat for persistent use.
